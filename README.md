# Install

Run:

```
rm -rf ~/.local/share/nvim
rm -rf ~/.local/state/nvim
rm -rf ~/.cache/nvim
rm -rf ~/.config/nvim
git clone https://github.com/marco-caamano/nvim-config.git ~/.config/nvim
```
